BeautifulSoup Self Project

Extracts information from WIKIPEDIA all animals page by specific conditions
Stores the information in a dictionary data collection
Downloads images of all animals from the web page, stores in locally in the same folder
Logs all of the steps in the way
Prints in the terminal and outputs to an HTML file

Instructions:

1. Dependecies required:
    pip install bs4
    pip install requests

2. To activate, run in cmd: "python3 main.py"
